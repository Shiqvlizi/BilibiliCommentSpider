# Bilibili 评论爬虫

看起来能用的版本.

使用方式:
1. 从 "[Releases](https://github.com/Shiqvlizi/BilibiliCommentSpider/releases)" 中下载最新版本
2. 解压压缩包
3. 编辑 VideoList.txt, 填入要爬取的视频 AV/BV 号, 一行一个
4. 打开 "BilibiliCommentSpider.exe", 并按提示操作
5. 所有爬取结果会保存到 "result.csv" 中

需要:
- 你的电脑上需要安装 .NET6 Runtime

基于:

- Microsoft.Playwright 浏览器自动化
- Spectre.Console 命令行输入

> 具体内容参见发行版的描述
